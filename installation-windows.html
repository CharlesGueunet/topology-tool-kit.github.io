Windows Installation
Please find below detailed instructions for the installation of TTK under Windows.

These specific notes describe the installation procedure for a Windows 10 operating system with Visual Studio 2015 (but the process should be very similar for other version). TTK is also supported on other operating systems:
 ·  Linux installation instructions (default target platform);
 ·  MacOS installation instructions.

1. Installing the dependencies

  • CMake (tested with 3.10.0)
  • Git for Windows (tested with 2.15.1)
  • QT5 (tested with 5.9.3)
  • Python (tested with 3.6.2)
  • Boost (tested with 1.65.1)
Please install all these applications with default settings and unpack Boost in C:\. Change the environment variable PATH to include the following paths:
 · C:\Program Files\Qt\5.9.3\msvc2015_64\bin
 · C:\Program Files\Python36

2. Installing VTK

Open the Command Prompt by pressing Windows+X to open the Power Users menu, and then click on Command Prompt (Admin).

Run the following commands to download the source code for ParaView and VTK (in the target locations C:\paraview and C:\paraview\VTK):
cd ../..
git clone https://gitlab.kitware.com/paraview/paraview.git
cd paraview
git submodule update --init --recursive

Run the following commands to configure VTK with CMake:
cd VTK
mkdir build
cd build
cmake-gui ../

Configure VTK (select generator "Visual Studio 14 2015 Win64" for x64-bit application).
Make sure to set the following variable:
 · CMAKE_CONFIGURATION_TYPES=Release

Click on the "Generate" button. Next, click on the "Open project" button.
Under Visual Studio, make sure to select the target named INSTALL.
Build it and close Visual Studio and CMake.

Change the environment variable PATH to include the path:
 · C:\Program Files\VTK\bin

3. Download TTK

Run the following commands for to download TTK sources to the target location C:\ttk:
cd ../../..
git clone https://github.com/topology-tool-kit/ttk.git

4. Patching ParaView

Run the following commands for patch the ParaView source tree on x86 installations:
cd c:\ttk\paraview\patch
patch-paraview-msvc-x86.cmd c:/paraview
cd ../../..

5. Installing ParaView

Run the following commands to configure ParaView with CMake:
cd paraview
mkdir build
cd build
cmake-gui ../

Configure ParaView. Make sure to set the following variables:
 · CMAKE_CONFIGURATION_TYPES=Release
 · PARAVIEW_ENABLE_PYTHON=ON
 · Qt5_DIR=C:\Program Files\Qt\5.9.3\msvc2015_64\lib\cmake\Qt5

Click on the "Generate" button. Next, click on the "Open project" button.
Under Visual Studio, make sure to select the target named INSTALL.
Build it and close Visual Studio and CMake.

Change the environment variable PATH to include the path:
 · C:\Program Files\ParaView\bin

6. Installing TTK

Run the following commands to configure TTK with CMake:
cd ../../ttk
mkdir build
cd build
cmake-gui ../

Configure TTK and make sure to set the following variables (check the Advanced checkbox to see all the available variables):
 · CMAKE_CONFIGURATION_TYPES=Release
 · Boost_INCLUDE_DIR=C:\boost_1_66_0\boost_1_66_0
 · ParaView_DIR=C:\paraview\build
 · Qt5_DIR=C:\Program Files\Qt\5.9.3\msvc2015_64\lib\cmake/Qt5
 · TTK_INSTALL_PLUGIN_DIR=C:\Program Files\ParaView\bin\plugins
 · TTK_ENABLE_OPENMP=OFF

Click on the "Generate" button. Next, click on the "Open project" button.
Under Visual Studio, make sure to select the target named INSTALL.
Build it and close Visual Studio and CMake.
