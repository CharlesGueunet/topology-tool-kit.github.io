Detailed Description

TTK VTK-filter that iteratively appends its input to a vtkMultiBlockDataSet.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.11.2018

This filter iteratively appends its input as a block to a vtkMultiBlockDataSet.

Parameters
InputvtkDataObject that will be added as a block (vtkDataObject).
OutputvtkMultiBlockDataSet containing all added blocks (vtkMultiBlockDataSet).
