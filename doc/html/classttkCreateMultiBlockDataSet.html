Detailed Description

TTK VTK-filter that stores up to 5 vtkDataObjects as blocks of a vtkMultiBlockDataSet.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
1.10.2018

This filter takes up to 5 vtkDataObjects and stores them as blocks of a newly created vtkMultiBlockDataSet.
