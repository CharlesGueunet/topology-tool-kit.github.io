Detailed Description

TTK VTK-filter that adds field data to a vtkDataObject.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
1.10.2018

This filter adds field data to a vtkDataObject based on a string and/or point/cell/field data of an optional vtkDataObject.

VTK wrapping code for the package.
