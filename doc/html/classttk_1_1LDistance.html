TTK lDistance processing package.

Author
Maxime Soler soler.nosp@m..max.nosp@m.ime@t.nosp@m.otal.nosp@m..com
Date
26/02/2017

LDistance is a TTK processing package that takes a scalar field on the input and produces a scalar field on the output.

See also
ttkLDistance.cpp for a usage example.

◆ setInputDataPointer1()

int ttk::LDistance::setInputDataPointer1 ( void *  data)
inline

Pass a pointer to an input array representing a scalarfield. The expected format for the array is the following: <vertex0-component0> <vertex0-component1> ... <vertex0-componentN> <vertex1-component0> <vertex1-component1> ... <vertex1-componentN> <vertexM-component0> <vertexM-component1> ... <vertexM-componentN>. The array is expected to be correctly allocated.

Parameters
dataPointer to the data array.
Returns
Returns 0 upon success, negative values otherwise.
See also
setVertexNumber() and setDimensionNumber().
