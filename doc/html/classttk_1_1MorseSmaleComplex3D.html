Detailed Description

TTK morseSmaleComplex3D processing package.

Class specialized in building the Morse-Smale complex of 3D triangulation.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
February 2017.

MorseSmaleComplex3D is a TTK processing package that takes a scalar field on the input and produces a scalar field on the output.

See also
ttk::Triangulation
ttkMorseSmaleComplex3D.cpp for a usage example.
