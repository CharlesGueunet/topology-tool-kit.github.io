Detailed Description

TTK VTK-filter for the computation of persistence curves.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Date
September 2016.

This filter computes the list of extremum-saddle pairs and computes the number of pairs as a function of persistence (i.e. the number of pairs whose persistence is higher than a threshold).

These curves provide useful visual clues in order to fine-tune persistence simplification thresholds.

Parameters
InputInput scalar field, either 2D or 3D, regular grid or triangulation (vtkDataSet)
Output0Table giving the number of persistent minimum-saddle pairs as a function of persistence (vtkTable)
Output1Table giving the number of persistent saddle-maximum pairs as a function of persistence (vtkTable)
Output2Table giving the number of persistent all extremum-saddle pairs as a function of persistence (vtkTable)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.
