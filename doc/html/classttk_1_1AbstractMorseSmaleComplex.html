Detailed Description

TTK abstractMorseSmaleComplex processing package.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
February 2017.

AbstractMorseSmaleComplex is a TTK processing package that takes a scalar field on the input and produces a scalar field on the output.

See also
ttk::Triangulation
vtkAbstractMorseSmaleComplex.cpp for a usage example.
