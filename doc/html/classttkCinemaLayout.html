Detailed Description

TTK VTK-filter that computes an image grid layout.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.10.2018

This filter computes a grid layout for images stored as blocks of a vtkMultiBlockDataSet.
