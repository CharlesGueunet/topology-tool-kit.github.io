Detailed Description

TTK processing package for the computation of persistence curves.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
September 2016.

This package computes the list of extremum-saddle pairs and computes the number of pairs as a function of persistence (i.e. the number of pairs whose persistence is higher than a threshold).

These curves provide useful visual clues in order to fine-tune persistence simplification thresholds.

See also
vtkPersistenceCurve.cpp for a usage example.
