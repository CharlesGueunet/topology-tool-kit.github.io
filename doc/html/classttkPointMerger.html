Detailed Description

TTK VTK-filter for point merging.

Author
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
February 2018.

This filter merges the points of a mesh whose distance is lower than a user defined threshold.

Parameters
InputInput data set (vtkDataSet)
OutputOutput data set (vtkDataSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.
