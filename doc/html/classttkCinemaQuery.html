Detailed Description

TTK VTK-filter that uses a SQL statement to select a subset of a vtkTable.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.09.2018

This filter creates a temporary SQLite3 database from the input table, performs a SQL query, and then returns the result as a vtkTable.

VTK wrapping code for the package.
