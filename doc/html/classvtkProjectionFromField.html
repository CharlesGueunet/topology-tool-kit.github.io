Detailed Description

TTK VTK-filter which projects a data-set to 2D given two point-data scalar fields to be used as 2D coordinates.

Author
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
November 2014.
Parameters
InputInput data-set, with at least two point data scalar fields or texture coordinates (vtkPointSet)
OutputOutput projected data-set (vtkPointSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.
